FAQ

Do I need to prefix tool calls?​

Yes. All inbound tool calls must be serverId:toolName so Broxy can route them deterministically.

What happens if a tool is missing?​

Missing tools, prompts, or resources are skipped with a warning. The rest of the preset still applies.

Can I change presets without restarting?​

Yes. Preset changes are applied immediately and the inbound facade is resynchronized.

Is HTTP SSE supported inbound?​

No. Use streamable HTTP for inbound traffic.
