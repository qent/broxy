Icon placeholders for Broxy installers

Provide platform-specific icons here (optional but recommended):

- macOS (DMG/APP): broxy.icns
  - Suggested sizes inside .icns: 16, 32, 64, 128, 256, 512, 1024
- Windows (MSI): broxy.ico
  - Suggested sizes inside .ico: 16, 32, 48, 64, 128, 256
- Linux (DEB/.desktop): broxy.png
  - Suggested size: 256x256 (can also include 512x512)

If these files are not present, packaging will continue without a custom icon.
